# Getting Started with Create React App

Root

yarn add concurrently

Backend

yarn add nodemon -D
yarn add apollo-server
yarn add graphql
yarn add mongoose
yarn add graphql-import
yarn add bcryptjs
yarn add jsonwebtoken

yarn run serve

Frontend

yarn add typescript
yarn add react-router-dom
